Material Design theme for WordPress.

== Description ==

This is a clean, modern and responsive theme focused on the content, based on the Material Design guidelines for Android App.

== Installation ==
	
1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.

== Frequently Asked Questions ==

= How to customize the theme ? =

You can easily change the colors, the logo and the header image by going to the WordPress customizer.

1. In your admin panel, go to Appearance > Customize.
2. Adjust settings and save.

= How to change the content of left sidebar ? =

You easily change the content of the left sidebar (the one who appear after clicking the &#9776;) :

1. In your admin panel, go to Appearance > Widgets.
2. Add widgets to the `Sidebar` and save.

= How to use custom formats ? =

This theme implement all the basic custom post formats of WordPress :

* aside - No title, only the content is displayed
* gallery - A gallery of images
* link - For linking to another site (can be only a title and an url as content, the theme will put the link to the title)
* image
* quote - A quote, not required to add a `blockquote`, It's done automatically
* status - A short note, will be contrasted
* video
* audio
* chat - Format a bubble chat, use `Username: text` and break line between every sentences

For using one, on the post editor, pick the appropriate format from the `Format` box and the theme will automatically format the content.

= How to setup custom menu ? =

You can change the content of the top right menu by create your own.

Create your custom menu :

1. In your admin panel, go to Appearance > Menu.
2. Add content (pages, ...) to the menu and save.

Attach the custom menu to the theme :

1. In your admin panel, go to Appearance > Customize.
2. On Navigation sub panel, choose your custom menu and save.
